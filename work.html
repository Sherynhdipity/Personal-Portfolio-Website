Student Management System

Student management system software, a program utilized to keep track of students. The primary goal in developing this software for students was to keep track of their academic progress and expenses. Only the system owner or administrator has access to this student management program.

A course administrator must first add the course name, duration, and fees to the organization's database. The next step is to build a teacher profile for the new hire with the appropriate course information. We may now add student information to our system after adding course and teacher details. In my project, you can search by course, student, fee status, and teacher. This program for student administration aids us in determining whether a student's fees have been paid in full

I did this project in C# Language, and SQL for the Database Language. I used Visual Studio 2019 for the IDE and Microsoft SQL Server 18 as the Database Engine.
